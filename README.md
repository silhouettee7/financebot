# financebot - бот для управления личным и групповым бюджетом

Financebot - Telegram-бот, который помогает вести личный и общий бюджет для группы людей. Он распределяет доходы на месяц, откладывает неиспользованные средства в накопления, строит диаграммы и таблицы расходов и предоставляет аналитику трат, также присутствует интеграция с API банков и LLM (моделями ИИ) для умных подсказок и автоматизации.

## Возможности

- Ведение личного бюджета.
- Общий бюджет для группы (семья, соседи, команда и т.п.).
- Автоматическое распределение месячного дохода по категориям.
- Перенос неиспользованных сумм в накопления.
- Формирование диаграмм, таблиц и графиков расходов и доходов.
- Анализ расходов: по категориям, по периодам, динамика и доли.
- Интеграция с API банков:
  - автоматический импорт операций;
  - сверка баланса и категорий.
- Интеграция LLM:
  - команды на естественном языке («на что я больше всего трачу в этом месяце?»);
  - умная категоризация операций;
  - персональные советы по экономии и планированию.

## Концепция

- Пользователь общается с ботом в Telegram.
- Бот сохраняет операции (доходы, расходы, переводы, накопления).
- Для групп создаются общие «кошельки» и правила распределения.
- Нерастраченные деньги в конце периода автоматически отправляются в накопления.
- По запросу бот строит отчёты и визуализации (диаграммы, графики, таблицы).
- После интеграции ИИ бот сможет объяснять, что происходит с бюджетом, «человеческим» языком.

## Типовые сценарии

- Вести личный бюджет и видеть, куда уходят деньги.
- Делить общие расходы в группе (квартира, семья, поездка).
- Планировать накопления и автоматически «скидывать» остатки в копилку.
- Смотреть отчёты по категориям и периодам.
- Спрашивать бота о состоянии бюджета и получать понятные ответы и рекомендации (после подключения LLM).
